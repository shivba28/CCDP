To Calculate Duration and Cost

  1. Click predict duration.
  2. Enter Builtup area, No of Floors, No of Workers and click predict.
  3. Entry will be shown in table.
  4. repeat step 2 to calculate for new configuration.
  5. Click on calculate under cost to calculate the cost of the project.
  6. select location, type of building, type and quality of work and click on calculate.
  7. Cost for the given configuration will be given considering the time factor of no of workers.
  8. It will also display the materials required.
  9. Click duration again from navbar to return to duration page.
  10. Here you'll observe the cost has been updated from TBC (To be Calculated) to the cost


    11. NOTE: *Use the navbar to switch between home, duration, cost and about us page*




To Calculate Only Cost

  1. Click predict cost
  2. Enter Builtup area, no of floors,location, type of building, type and quality of work and click on calculate.
  3. This will display the Cost for the given configuration.
  4. It will also display the materials required.

Methodology

  1. We gathered data required for constuction schedule prediction and raw material calculation as well as cost for the projects.
  2. After analysing the relations between the data we created the final datasets by performing data manipulations.
  3. We first created the "Duration prediction" model by using the dataset and testing it on various algorithms. We selected "Multiple Linear Regression" as our final model.
  4. We then created the "Cost prediction and raw material calculator" module by using the data obtained in our prior phases.
  5. We then created the "hybrid Cost prediction" module which uses the time factor to predict the cost.
  6. Then the modules were deployed on the web using Python Flask and front end technologies.
  7. Database connectivity was added for helping create more data for further research purposes.
